## How to Use
1. **Clone the repository**:
   ```bash
   git clone https://github.com/shaikat-hazra/portfolio.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run locally**:
   ```bash
   npm start
   ```
4. **Build for production**:
   ```bash
   npm run build
   ```
